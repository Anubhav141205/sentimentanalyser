# Multimodal Sentiment Analysis

This project implements a Multimodal Sentiment Analysis system using:

- BERT (Text Encoder)
- ResNet-50 (Image Encoder)
- Late Fusion Technique
- Modality Dropout
- Streamlit Web Application

## Features
- Text-only prediction
- Image-only prediction
- Combined multimodal prediction
- Handles missing modalities

## Tech Stack
- Python
- PyTorch
- Transformers (HuggingFace)
- Streamlit
- Scikit-learn

## How to Run

```bash
pip install -r requirements.txt
streamlit run app.py
